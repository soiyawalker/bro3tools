# ブラウザ三国志向けツール群

個人的にメンテしているブラウザ三国志向けツール群です。

* bro3_hide_ad.user.js
  * 広告を非表示にするツールです
* bro3_pretty_skill
  * スキル実行を助けるツールです
  * デッキ画面で対応のスキルをクリックするとスキルを実行します
  * 回復系スキルと鹵獲系スキルに対応しています
